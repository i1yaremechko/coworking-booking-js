# Smart Coworking Booking System (Vanilla JS)

Навчальний проєкт з розробки інтерактивної системи бронювання робочих місць (коворкінгу). Побудований на модульній архітектурі з використанням чистого JavaScript та інтеграцією з REST API.

## Live Demo
[Переглянути проект](https://i1yaremechko.github.io/coworking-booking-js/)

## Технології
- **Vanilla JavaScript** (ES6+ Modules)
- **SCSS** (Sass)
- **ESLint / Prettier** (Airbnb Style Guide)
- **LocalStorage**

## Функціонал
- Динамічна сітка: Відображення 5 робочих столів з погодинним розкладом.
- Керування бронюваннями: Повноцінна можливість створювати, редагувати та видаляти записи в реальному часі.
- Серверна синхронізація: Всі дані зберігаються на зовнішньому сервері, що забезпечує їх сталість після оновлення сторінки.
- Розумна валідація:
  - Перевірка на перекриття часу (overlap): неможливо забронювати стіл, який уже зайнятий.
  - Кратність часу: бронювання лише на повні години.
  - Валідація тривалості: час закінчення не може бути раніше часу початку.

## Як запустити
1. Клонуйте репозиторій: git clone https://github.com/i1yaremechko/google-calendar-js.git
2. Встановіть залежності: `npm install`
3. Запустіть компіляцію стилів: `npm run build:css`
4. Використовуйте `Live Server` для перегляду `index.html`.
