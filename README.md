# Smart Coworking Booking System

Навчальний проєкт із розробки системи бронювання робочих місць у коворкінгу. Побудовано на базі архітектури Google Calendar.

## Технології
- **Vanilla JavaScript** (ES6+ Modules)
- **SCSS** (Sass)
- **ESLint / Prettier** (Airbnb Style Guide)

## Як запустити
1. Встановіть залежності: `npm install`
2. Запустіть компіляцію стилів: `npm run build:css`
3. Використовуйте `Live Server` для перегляду `index.html`.
